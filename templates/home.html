Welcome to the Home Page

This page provides an explanation of the routes in the Flask application:

/api/v1/similar?word=XXX GET:

Find similar words for given word

/api/v1/add-word - POST:

Add a word to the dictionary The request body should be a JSON object as follows: { "word":"[word_to_add]>" }

/api/v1/stats - GET:

Get general statistics about the word querying

/close-connection - POST:

Close the database connection
